## How to Install *Where Winds Meet Mods* 

Installing mods can be a bit tricky for first-timers, but don’t worry. Below is a simple guide to help you install mods for *Where Winds Meet*:

### 1. Download the Mod Files
Visit a trusted modding site like Nexus Mods or the *Where Winds Meet* community forums to download your chosen mod. Ensure the mod is compatible with your version of the game to avoid any issues during installation.

### 2. Install a Mod Manager
While it’s possible to install mods manually, using a mod manager like Vortex (available on Nexus Mods) can simplify the process. It allows you to manage, install, and uninstall mods with a few clicks.

### 3. Backup Your Files
Before you install any mods, it’s always a good idea to backup your game files. This ensures you can revert back to the original version of the game if something goes wrong.

### 4. Install the Mod
With a mod manager, installing mods is usually as simple as dragging and dropping the mod into the manager. The software will automatically place the files in the correct directory.

If you’re installing mods manually, you’ll need to navigate to the game's installation folder and copy the mod files into the designated folder (usually “Mods” or “Data”).

### 5. Test the Mod
Once installed, launch *Where Winds Meet* and check if the mod is working as expected. If the game crashes or the mod isn’t functioning correctly, check the mod's documentation for troubleshooting tips or compatibility issues.
